## 包的安装更新

- 更新所有R包


```r
#安装包
install.packages("rvcheck")
#加载包
library(rvcheck)
#检查R是否有更新
rvcheck::check_r()
# 更新所有R包
rvcheck::update_all(check_R = FALSE,which = c("CRAN","BioC","github")
```

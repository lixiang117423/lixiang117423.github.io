## FastTree构建进化树

[FastTree](http://www.microbesonline.org/fasttree/)是基于最大似然法构建进化树的软件，使用简单。Windows端直接下载，完后运行即可：

```shell
.\FastTree.exe .\YourAlignResults.fasta
```

前面是软件名称，后面是比对好的序列文件。

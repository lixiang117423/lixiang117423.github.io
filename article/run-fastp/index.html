运行fastp的脚本


一个高效、自动化的 FASTQ 文件批量质控工具包，基于强大的 fastp 工具。
An efficient and automated toolkit for batch quality control of FASTQ files, built upon the powerful fastp tool.

概览 | Overview

fastp_toolkit 是一个 Python 脚本工具包，旨在简化和自动化使用 fastp 对双端测序（Paired-End）的 FASTQ 数据进行质量控制的流程。它能够自动发现指定目录下的样本对，并行处理，并生成清晰的日志和总结报告，极大地提高了生物信息分析流程的效率和可重复性。

fastp_toolkit is a Python script package designed to simplify and automate the quality control workflow for paired-end FASTQ data using fastp. It automatically discovers sample pairs in a specified directory, processes them in parallel, and generates clean logs and summary reports, significantly improving the efficiency and reproducibility of bioinformatics analysis pipelines.

✨ 特性 | Features

  • 🚀 批量处理 (Batch Processing): 自动查找并处理输入目录下的所有 FASTQ 样本对。
  • ⚙️ 高度可配 (Highly Configurable): 通过命令行参数或配置文件轻松调整 fastp 的所有核心质控参数。
  • 📑 自动报告 (Automated Reporting): 为每个样本生成详细的 HTML 和 JSON 质控报告，并在处理结束后生成一份总的总结报告。
  • 📄 详细日志 (Detailed Logging): 记录详细的运行日志，包括执行的命令、处理状态和错误信息，方便追踪和调试。
  • 🧩 模块化设计 (Modular Design): 代码结构清晰，易于维护和扩展。既可以作为命令行工具使用，也可以作为库集成到其他 Python 项目中。
  • ✔️ 智能检查 (Smart Checks): 自动检查 fastp 可执行文件是否存在，并跳过已处理完成的样本，支持任务中断和续跑。
  • 🌐 双语支持 (Bilingual Support): 代码注释和输出信息均提供中英双语，便于不同语言背景的用户使用。

📋 依赖与环境 | Prerequisites

  • Python: Python 3.7+
  • fastp: 需要预先安装 fastp 工具，并确保其在系统的 PATH 环境变量中，或者通过 --fastp-path 参数指定其可执行文件路径。
    • fastp GitHub 仓库

🛠️ 安装 | Installation

  1. 准备文件:
    该工具包以打包文件的形式提供。您需要一个 extract_modules.py 脚本（或手动操作）来将代码提取到正确的目录结构中。

  2. 创建项目结构:
    请按照以下结构组织您的文件：

    • 将打包文件中 FILE: fastp_toolkit/.... 的内容分别保存到 fastp_toolkit 目录下的对应文件中。
    • 将 FILE: run_fastp.py 的内容保存为 run_fastp.py。
    • 创建一个 raw_data 目录来存放您的原始测序数据。

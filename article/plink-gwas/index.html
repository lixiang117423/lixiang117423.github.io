plink做GWAS的脚本

完整的PLINK质量性状GWAS分析流程，支持非标准染色体编号和群体分层控制
Complete PLINK qualitative trait GWAS analysis pipeline with support for non-standard chromosome naming and population stratification control

🌟 主要特点 | Key Features

  • 🧬 支持非标准染色体编号 - 兼容OV、KY等非人类物种的染色体命名方式
  • 📊 完整的质量控制流程 - 个体缺失率、SNP缺失率、MAF、HWE检验
  • 🔬 群体分层控制 - LD剪枝 + 主成分分析校正群体结构
  • ⚡ 自动化流程 - 从VCF文件到可视化结果的一站式分析
  • 📈 内置可视化 - 自动生成Manhattan图和QQ图
  • 📋 详细报告 - 生成包含统计信息和结果解释的分析报告
  • 🔧 模块化设计 - 清晰的代码结构，易于维护和扩展

必需软件 | Required Software

  • Python 3.7+

  • R 4.0+ - 用于生成可视化图形
  • R包: qqman - 用于Manhattan图和QQ图

📖 详细使用说明 | Detailed Usage

命令行参数 | Command Line Arguments

必需参数 | Required Arguments

  • -v, --vcf-file - VCF文件路径 (支持.gz压缩)
  • -p, --phenotype-file - 表型文件路径

可选参数 | Optional Arguments

输出设置 | Output Settings

  • -o, --output-dir - 输出目录 (默认: plink_gwas_results)

质量控制 | Quality Control

  • --mind - 个体缺失率阈值 (默认: 0.05)
  • --geno - SNP缺失率阈值 (默认: 0.05)
  • --maf - 最小等位基因频率 (默认: 0.01)
  • --hwe - Hardy-Weinberg平衡P值阈值 (默认: 1e-6)

LD剪枝 | LD Pruning

  • --ld-window-size - LD窗口大小，kb (默认: 50)
  • --ld-step-size - LD步长，SNP数 (默认: 5)
  • --ld-r2-threshold - LD r²阈值 (默认: 0.2)

主成分分析 | Principal Component Analysis

  • --pca-components - 计算的主成分数 (默认: 10)
  • --pca-use - 关联分析中使用的主成分数 (默认: 5)

显著性阈值 | Significance Thresholds

  • --significance-threshold - 全基因组显著性阈值 (默认: “5e-8”)
  • --suggestive-threshold - 提示性关联阈值 (默认: “1e-5”)

计算资源 | Computing Resources

  • --threads - 线程数 (默认: 1)

📁 输出文件说明 | Output Files Description

主要结果文件 | Main Result Files

文件名 描述 内容
analysis_report.txt 分析总结报告 样本统计、参数设置、结果汇总
gwas_results_ADD.txt 主要关联分析结果 所有SNP的关联分析统计量
significant_hits.txt 显著关联位点 P < 5e-8的SNP
suggestive_hits.txt 提示性关联位点 P < 1e-5的SNP
manhattan_plot.png Manhattan图 全基因组关联信号可视化
qq_plot.png QQ图 P值分布和质量评估

中间文件 | Intermediate Files

文件名 描述
data_qc1.* 质控后的PLINK格式数据
pca_with_header.txt 主成分分析结果
chromosome_mapping.txt 染色体编号映射关系
plink_gwas_analysis.log 详细运行日志

结果文件详解 | Detailed Results Description

gwas_results_ADD.txt 主要列说明：

  • CHR: 染色体
  • SNP: SNP标识符
  • BP: 碱基对位置
  • A1: 效应等位基因
  • OR: 优势比 (Odds Ratio)
  • P: P值
  • L95/U95: 95%置信区间

结果解释 | Results Interpretation：

  • OR > 1: 该等位基因增加感病风险
  • OR < 1: 该等位基因具有保护作用（促进抗病）
  • P < 5e-8: 达到全基因组显著性水平
  • P < 1e-5: 达到提示性关联水平

⚠️ 注意事项 | Important Notes

数据要求 | Data Requirements

  1. VCF文件格式: 必须符合VCF 4.0+标准
  2. 表型编码: 0=对照/抗病，1=病例/感病，缺失值用NA表示
  3. 样本匹配: VCF和表型文件中的样本ID必须一致
  4. 染色体命名: 支持标准编号(1,2,3…)和非标准编号(OV01,KY_chr1等)

质量控制建议 | QC Recommendations

参数 推荐值 严格值 宽松值
--mind 0.05 0.02 0.10
--geno 0.05 0.02 0.10
--maf 0.01 0.05 0.005
--hwe 1e-6 1e-8 1e-4

样本量建议 | Sample Size Recommendations

  • 最小样本量: 100个个体 (50病例 + 50对照)
  • 推荐样本量: 500+个个体 (250病例 + 250对照)
  • 理想样本量: 1000+个个体 (500病例 + 500对照)

性能优化建议 | Performance Optimization

  1. 使用多线程: --threads 8 (根据CPU核心数调整)
  2. 压缩VCF文件: 使用.gz压缩减少I/O时间
  3. 调整内存: 大数据集考虑使用PLINK2.0
  4. 分块分析: 超大数据集可按染色体分块处理
